HTML

HTML (Hypertext Marked Language) 即 超文本标记语言。

扩展名有两种:分别是 .htm 和 .html    例：1.html

web标准分三层架构: Html结构 css表现 js 行为。

单标签:

<标签 />

<标签 属性=”值” />

双标签:

<标签>< / 标签>

<标签 属性=”值”>< / 标签>

嵌套标签：

<标签1><标签2>< / 标签2>< / 标签1>

CSS

Css是Cascading Style Sheet 的缩写，中文即“层叠样式表”用于装饰，美化网页,布局和外观显示样式

CSS的扩展名为：.css    例：1.css

CSS加入到html网页里面有三种方式

1. 行内式 2. 内部样式表 3. 外部样式表

CSS的基础选择器有四种

1.id选择器 2.标签选择器 3.类选择器 4.通配符选择器

CSS的复合选择器有五种

1.后代选择器 2.子元素选择器 3.交集选择器 4.并集选择器   5.链接伪类选择器
